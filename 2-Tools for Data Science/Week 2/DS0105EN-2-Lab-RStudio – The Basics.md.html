Objective for Exercise:

This lab introduces you to R and RStudio

Exercise:

  1. For this lab, login to RStudio Skills Network Lab here

  2. Start RStudio. An R-Studio window will open up.

  3. Click in the tiny plus symbol top left and select R Script

  1. An untitled R Script panel opens. It would look like this.
  1. Now we load the iris dataset. Please enter the following lines into the editor window which just appeared. Then select them all such that they turn blue. Then click on the tiny Run icon just above the editor window.
library (datasets)
data(iris)
View(iris)
  1. You are directly taken to the data view tab to inspect your dataset. We can see that there are five columns in this data set and that the first four are floating point and the last one is label of data type string which contains the category value of our data set. We also see that we have 150 entries in total of which we are seeing the first 19.
  1. Now let's find how many different species there are present in the data set. Type the following command into the editor window and click the run.
unique(iris$Species)

In the Console window at the bottom you’ll see the result of the executed command and will know that there are only three different species present in the data set.

  1. Now it’s time to look into the data set in more detail.

  2. Open a new Terminal.

  1. Run the following commands in the terminal, one after the other to download the packages we need to install.
curl https://cran.r-project.org/src/contrib/hms_0.5.3.tar.gz --output hms_0.5.3.tar.gz 

curl https://cran.r-project.org/src/contrib/progress_1.2.2.tar.gz --output progress_1.2.2.tar.gz   

curl https://cran.r-project.org/src/contrib/reshape_0.8.8.tar.gz  --output reshape_0.8.8.tar.gz  

curl https://cran.r-project.org/src/contrib/GGally_2.0.0.tar.gz --output GGally_2.0.0.tar.gz

  1. After downloading the packages, go to Console.

  2. Go to Tools > Install Packages. From Install from dropdown select Package Archive File (.tar.gz) option.

  1. Now, click Browse for Package archive.

  2. Select resources > rstudio > rpackages > hms_0.5.3.tar.gz.

In the same way,

Install progress_1.2.2.tar.gz

Install reshape_0.8.8.tar.gz

Install GGally_2.0.0.tar.gz

This concludes this lab, I hope you’ve enjoyed it!
